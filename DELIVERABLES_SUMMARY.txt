DATABASE DOCUMENTATION

Properties Table (173,743 records)
  - UUID primary key
  - County parcel ID (NULL for ~2,697)
  - Address, city, ZIP information
  - Owner name
  - Market, appraised, taxable values (ALL IN CENTS)
  - PostGIS geometry (WGS84 coordinates)
  - Subdivision foreign key
  - Audit timestamps

Subdivisions Table (4,041 records)
  - UUID primary key
  - Subdivision name
  - Polygon boundary geometry
  - Audit timestamps

CRITICAL DATA NOTES
  - All monetary values are in CENTS (must divide by 100 for dollars)
  - Geometry uses WGS84 (EPSG:4326) standard lat/lon coordinates
  - ~2,697 properties have NULL parcel_id (use UUID id instead)
  - Not all properties assigned to subdivisions

TEAM SETUP PROCESS
================================================================================

TIME TO READY: ~5 minutes

1. Clone & Setup                           (30 seconds)
   git clone <repository-url>
   cd taxdown

2. Virtual Environment                     (1 minute)
   python -m venv venv
   source venv/bin/activate      # Mac/Linux
   venv\Scripts\activate         # Windows

3. Install Dependencies                    (2 minutes)
   pip install -r requirements.txt

4. Configure Database                      (1 minute)
   cp .env.example .env
   [Edit .env with DATABASE_URL from MJ]

5. Test Connection                         (30 seconds)
   python src/utils/test_connection.py
